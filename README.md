Plug n Play
Just drop in your project.
For java do:
```
  public ClassName(Main plugin) {
    this.plugin = plugin;
    // constructor
  }
```
Or for kotlin:

```
private val plugin: Main? = null
```
and then use `plugin.getConfig().getString("name in config.yml")` to use the config manager.
